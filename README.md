# Soletude-Footwears
It is a final semester project in which our team has created a website of business startup e-commerace named as "Soletude", using html and css. In this repository i've uploaded the website codes and all the related data stored in excel sheet along with a readme file that contains pictures of website and links of word report, ppt &amp; excel datasheet.
Our project is about a business startup agency. so we chose e-commerace website as a business startup of footwears. We have made a partially responsive website of a brand with multiple pages of different categories along with product images, user accounts and cart. Moreover, we have oragnized the data of our brand in an excel data workbook with an overview of products' details, monthly sales, employees info, advertisement costs, expenses and all the profits or loss. along with that we have also highlighted the data of customer feedback and most demanding products according to their sales. we have presented all the project details in a PPT file and a word report and showcased and overview of website in an animated video.
